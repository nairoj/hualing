## 4. 已确认约束（MVP）
- 平台：PC。
- 语言：GDScript。
- 难度：仅标准。
- 关卡：房间、户外、地下一层。
- 撤离点：每层 1 个固定点。
- 失败惩罚：死亡丢失 50% 本次资源。
