## 1. 项目概述
### 1.1 一句话描述
- 将柜子、蜡烛、小狗化灵为伙伴，在昼夜压力下规划时间，尽量白天闯关并安全撤离。

### 1.2 游戏类型与平台
- 类型：动作冒险（非 Roguelite）
- 视角：2D 素材 + 伪 2.5D 表现（front-top）
- 平台：`PC（MVP）`；手机/Web 后续适配（Godot 导出验证后推进）
- 输入：`键鼠（MVP）`；触屏方案后续再定（不支持手柄）

### 1.3 目标用户
- 核心玩家画像：轻度硬核动作冒险玩家，偏好元气骑士式战斗强度，但希望有明确关卡推进而非纯肉鸽随机
- 竞品参考：元气骑士（战斗节奏参考）、泰拉瑞亚（道具驱动成长参考）
- 预期单局时长：12 分钟
- 预期通关时长：8 小时

### 1.4 设计原则（最多 5 条）
1. 战斗驱动优先：核心体验以战斗闯关为主（约 70%）。
2. 组合策略来自伙伴搭配，而非复杂 build 系统。
3. 玩家成长由道具与伙伴提供，不采用角色等级升级。
4. 时间规划是核心策略：鼓励玩家在白天推进，夜晚以撤离保收益为主。
5. 先保证 PC 端核心体验闭环，再做移动端与 Web 适配。

## 2. 世界观与叙事
### 2.1 世界法则
- 万物有灵的原因：设定直给，不在 MVP 阶段解释底层因果。
- 化灵的限制：MVP 仅允许固定 3 个对象（柜子、蜡烛、小狗）。
- 主角为何能化灵：设定上主角是当前唯一可稳定化灵者，不强调故事合理性。

### 2.2 主线结构
- 开场：玩家从房间床上醒来；初始有 2 把破剑、3 个化灵散，房间内有 1 个蜡烛与 1 个柜子可互动。
- 目标：将 3 个固定道具（柜子、蜡烛、小狗）化灵为伙伴，离开房间经过户外，在地下入口完成伙伴编队后进入地下一层打怪推进。
- 阶段节点（至少 3 个）：
  1. 房间内完成资源确认与化灵。
  2. 户外集合并在入口完成拖拽编队。
  3. 进入地下一层，清理战斗并击败精英怪。
- 结局条件：击败大史莱姆与当层全部小怪后通关第一层；点击下一层入口后临时回到房间。

## 3. 核心循环
### 3.1 一次探索流程
1. 家园准备：配置道具与伙伴，评估本次探索目标层。
2. 选层进入：通过电梯进入目标层；已通关楼层可直达回刷。
3. 白天推进：以战斗与轻解谜推进，寻找通往下一层入口或关键敌人。
4. 黄昏抉择：怪物压力线性上升，决定继续推进还是准备撤离。
5. 夜晚高压：刷怪效率提升至 3 倍，生存压力显著上升。
6. 撤退/死亡结算：仅可在固定撤离点撤离（MVP 每层 1 个）；玩家死亡则本次探索失败。
### 3.2 关键抉择点
- 进入前：选择本次挑战层、伙伴组合、携带撤离/恢复类道具。
- 中途：根据时间段动态决策；白天优先推进目标，黄昏开始评估撤离路径。
- 退出时：夜晚高压下优先撤退保收益，避免无意义硬拼。

### 3.3 成功失败判定
- 本层成功条件：混合条件；MVP 阶段定义为“击败第一层指定精英怪”。
- 探索失败条件：玩家死亡。
- 强制撤退条件：无全局强制；撤退行为受规则限制（仅固定撤离点，MVP 每层 1 个）。
## 4. 化灵系统
### 4.1 对象分类
- MVP 可化灵物品：固定 3 种（柜子、蜡烛、小狗）。
- 不可化灵规则：除上述 3 种外，MVP 阶段其余物品均不可化灵。

### 4.2 化灵流程
- 背包物品化灵流程：选择目标物品 -> 使用化灵散 -> 播放化灵特效 -> 生成对应伙伴。
- 场景物品化灵流程：MVP 暂不开放（只做固定对象化灵）。
- 视觉表现与反馈：化灵后原物品消失，伙伴实体进入队伍；家园化灵物品夜晚在屋外帐篷休息。

### 4.3 成本与风险
- 消耗资源：MVP 初始直接拥有化灵散（可完成 3 个固定对象化灵）。
- 成功率：MVP 固定 100%。
- 失败后果：MVP 不存在化灵失败。
- 冷却机制：MVP 不设冷却。
- 唯一性规则：一个物品化灵后即转化为伙伴，不可重复化灵。

### 4.4 家园取舍后果
- 家具被化灵后家园功能损失：被化灵物品从原位置永久移除。
- 替代方案（临时/永久）：MVP 不提供召回或还原机制。
- 舒适度与休息恢复公式：当前版本回家后玩家生命值回满；被化灵伙伴在屋外帐篷休息。

## 6. 战斗系统
说明：详细数值请见 `docs/numeric_system_design.md`。

### 6.1 玩家基础属性（MVP 初版）
| 属性 | 初始值 | 说明 |
|---|---:|---|
| 生命值 | 100 | 归零则本次探索失败 |
| 攻击力 | 12 | 影响近战与部分技能伤害 |
| 防御力 | 4 | 用于伤害减免 |
| 移速 | 130 | 基础移动速度 |
| 暴击率 | 5% | MVP 基础暴击 |

### 6.2 战斗与伙伴规则
- 玩家武器类型：固定近战。
- 远程副武器：MVP 不开放。
- 阿奇行为：自动跟随主角，在主角周围主动攻击敌人（小狗伙伴）。
- Buff 共享：阿奇与玩家共享当前增益效果。
- 蜡烛灵增益：编队中存在蜡烛灵时，玩家与阿奇获得常驻 `+10%` 伤害加成，并扩大视野。
- 柜子灵功能：提供背包扩容能力（默认 2 格 -> 携带后 6 格）。
- 基础伤害公式（简版）：`最终伤害 = max(1, 攻击力 * 技能倍率 - 防御力)`。

### 6.3 状态效果表（MVP）
| 状态 | 类型 | 数值效果 | 持续时间 | 叠加规则 |
|---|---|---|---|---|
| 增伤（蜡烛） | 正面 | 伤害 +10% | 常驻（编队生效） | 不叠加 |
| 其他异常状态 | - | MVP 暂不启用 | - | - |

### 6.4 敌人与 Boss
- 第一层基础敌人（3-5种）：
  - 小史莱姆：近战贴身，低血量。
  - 洞穴蝙蝠：短冲刺骚扰，中低血量。
- 第一层精英怪：大史莱姆（关卡关键目标）。
- Boss 阶段切换条件：MVP 暂不启用多阶段 Boss。
- 技能与可读性：
  - 大史莱姆核心技能为“跳砸”。
  - 其余强度主要通过更大体型与更高攻击数值体现。

### 7.2 昼夜系统
- 一天总时长：24 分钟（现实时间）。
- 白天时长：12 分钟（06:00 - 18:00）。
- 黄昏时长：2 分钟（18:00 - 20:00）。

- 夜晚增压规则（刷怪、攻防倍率等）：
  - 白天倍率 1.0。
  - 黄昏阶段怪物刷新倍率从 1.0 线性提升到 3.0。
  - 夜晚怪物刷新倍率固定 3.0。
  - 敌人攻防倍率暂不提升（后续平衡再评估）。

### 7.3 难度节奏
- 前 10 分钟体验目标：完成一次基础探索教学并理解“黄昏撤离”抉择。
- 30 分钟体验目标：可稳定完成 1 层探索，并尝试不同伙伴组合。

## 12. MVP 定义与验收
### 12.1 MVP 目标
- 实现完整核心动线：玩家从房间醒来 -> 到户外编队 -> 下地下一层 -> 完成战斗并击杀大史莱姆后可撤离。

### 12.2 MVP 内容清单
- 玩家：可移动、固定近战攻击、生命结算。
- 场景：房间、户外、地下一层。
- 敌人：基础小怪 + 第一层精英怪（大史莱姆）。
- 可化灵物品：柜子、蜡烛、小狗（3 个固定对象）。
- 初始道具：背包内 2 把破剑、3 个化灵散；出生房间内 1 个蜡烛、1 个柜子。
- 编队与入口：地下洞穴入口前可选择随行伙伴，确认后进入关卡。
- UI界面：基础 HUD（含时段与倒计时）、拖拽编队、化灵确认框、战斗结算。

### 12.3 验收标准（必须量化）
- 可玩性验收：可稳定完成 `房间 -> 户外 -> 地下一层` 动线，并能完成 3 个物品化灵。
- 稳定性验收：连续游玩 30 分钟无崩溃、无阻断流程的严重 Bug。
- 战斗验收：可正常刷怪并击杀第一层精英怪（大史莱姆）。
- 体验验收：玩家能理解“白天推进、夜晚高压、优先撤离保收益”的策略导向。
